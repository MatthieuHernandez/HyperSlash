<h1 align="center">Hyper Slash</h1>


**Hyper Slash** is a fast-paced hack 'n' slash game focused on:

- Defeat enemies with style and precision
- Far too fast movements
- Score as many points as possible *(Scoring system still in development)*
- Hard-hitting music *(Hyperpop in coming)*

Powered by:
- Open-source development
- Unreal Engine 5.7
- Modern C++20

## Development

### Setup

1. Clone the repository
2. Right click on `HyperSlash.uproject`
3. Click `Generate Visual Studio project files`
4. Open `HyperSlash.sln`
5. Build the project
6. Launch the project with Unreal Engine 5.7
